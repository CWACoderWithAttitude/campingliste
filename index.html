Camping List

When preparing for a camping trip there's a lot of stuff to remember. We tried lists analog and digital.
We tried paper and digital. Either way had pros and cons.
This is the latest iteration.

How to use

Just edit the markdown files and run ./convert.sh. The script uses pandoc and asciidoctor to generate pdfs.
You don't need to have either installed locally - the script runs them from docker containers.

It will produce printable pdfs with lists where you can tick each item once it's done.

Pull requests with updated lists or script improvements are encouraged and very welcome.
